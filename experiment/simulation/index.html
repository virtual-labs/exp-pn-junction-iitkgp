Instructions

    1. Set Initial Parameters:
      • NA: Set the acceptor concentration in the p-region (e.g., 1e15 /cm3).
      • ND: Set the donor concentration in the n-region (e.g., 1e15 /cm3).
      • V: Adjust the applied voltage (e.g., -0.5 V).
      • T: Set the temperature (e.g., 300 K).
    2. Submit Parameters:
      • Click the "Submit" button to apply the set parameters.
    3. View Results: The simulation will generate the following graphs:
      • Band Diagram: Displays the energy bands across the pn junction.
      • Current-Voltage Characteristics: Shows the I-V curve of the diode.
      • Charge Density: Illustrates the charge distribution across the junction.
      • Electric Field: Displays the electric field intensity in the depletion region.
      • Carrier Densities: Shows the concentration of electrons and holes.
      • log(Carrier Densities): Provides a logarithmic view of the carrier concentrations.
    4. Analyze Data: Use the graphs to analyze the electrical properties of the pn junction in depletion mode. Note the following:
      • Depletion Width (W): Measure the width of the depletion region.
      • Built-in Potential (Vbi): Determine the built-in potential of the junction.
      • Electric Field (E): Examine the electric field distribution.
